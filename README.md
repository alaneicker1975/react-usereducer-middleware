# Atomikui Starter Project

### Setup

```
npm install
```

### Running the project

```
npm run start
```

### Running Unit Tests

Unit tests are designated with a `.test.js` extension.

```
npm run test:unit
```

### Running Accessibility Tests

Accessibility tests are designated with a `.spec.js` extension to separate them from unit tests.

```
npm run test:unit
```

### Linting

JavaScript

```
npm run lint:js
```

CSS

```
npm run lint:css
```
